# shopyfyEcommerce
An Ecommerce website for various stylish clothes
## Technologies used:
* React Js
* React useContext + useState
* React Router
* Vanila css with flex,grid and media query
* Toastity
# Features
* Home Page
* Product Listing Page
* Single Product Page
* various Filters
   * Sort By Rating
   * Sort By Category
   * Sort By Price
* Wishlist Management
* Cart Management
* Loading and alerts
* Login Page
* Error Page
* Responsive UI
